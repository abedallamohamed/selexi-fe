# Selexi Frontend

Applicazione Vue 3 + Vite con Docker.

## Setup e Avvio

```bash
# Avvia l'applicazione
docker compose up -d --build

# Visualizza i log
docker logs selexi-fe

# Ferma l'applicazione
docker compose down
```

## Accesso

- Applicazione: http://localhost:5173
- Vue DevTools: http://localhost:5173/__devtools__
